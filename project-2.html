Suspension

Project Overview

Barrel down a trecharous mountain while a mysterious, relentless entity persues you. Suspension is a fast-paced biking adventure where you ride through dense forests, rugged terrains, and scenic trails. The player must react to dynamic hazards like falling trees, winding paths, and blazing forest fires. A prodecurally generated track ensures each run is unique, with a difference suite of challenges to overcome each time.

Suspension was developed as part of an intership with game studio Serenity Forge. Working with 5 other junior developers, I developed a minimum viable product and demo to be presented as a pitch deck to mentors and other figures in the gaming industry. My personal contribution were the volumetric fog shader, "entity" chasing mechanics (foliage decay, darkness effect, dyanamic speed settings and chasing), bug ticketing and solving.

A build/code link is unavailable for this project, in compliance with an NDA with Serenity Forge Games.
